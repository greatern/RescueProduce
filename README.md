# RescueProduce– Full-Stack Food Rescue Platform | (Final Year Project) team of 3

**Description: Led the full-stack development of a cross-platform mobile and web
application designed to reduce food waste and insecurity in South Africa. The
platform connects supermarkets with surplus food to NGOs and volunteers for
efficient redistribution, featuring real-time notifications, demand matching, and
logistics coordination.**

• Tech Stack: React Native (Expo Go)
• TypeScript • Express.js • React • Vite • Tailwind CSS • Docker
• Key Achievements:
o Engineered a cohesive ecosystem with a React Native/TypeScript mobile
app for donors and volunteers and a React/Vite/TypeScript admin web
portal.
o Developed a robust, scalable RESTful API backend
using Node.js and Express.js.
o Containerized the entire application stack using Docker for consistent
development and deployment environments.
o Implemented Tailwind CSS across both web and mobile (via NativeWind)
to ensure a consistent, responsive, and modern UI.
